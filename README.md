# PHP API Response Toolkit

A lightweight, framework-independent PHP library for creating consistent and standardized JSON API responses.

## Purpose

Provide a simple, reusable, and framework-independent way to generate
consistent JSON API responses across PHP applications.

## Features

- Standard success responses
- Standard error responses
- Validation error responses
- HTTP status helpers
- Framework independent
- PSR-4 autoloading
- Lightweight and reusable

## Requirements

- PHP 8.1+

## Installation

```bash
composer require sharadkashyap/php-api-response-toolkit
```

## Project Status

This project is currently under active development.
The initial public release will focus on providing a lightweight,
framework-independent API response toolkit for PHP.
